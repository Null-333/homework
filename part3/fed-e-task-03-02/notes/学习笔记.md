### 通过阅读源码，回答下面代码在页面上的输出结果
同时设置render和template会渲染哪一个？
```javascript
const vm = new Vue({
  el: '#app',
  template: '<h3>Hello template</h3>',
  render(h) {
    return h('h3', 'Hello render'),
  },
});
```
阅读源码记录:
- el 不能是 body 或者 html 标签
- 如果没有 render，把 template 转换成 render 函数 
- 如果有 render 方法，直接调用 mount 挂载 DOM
